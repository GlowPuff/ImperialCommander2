<color="red">Защитные протоколы:</color>
Каждый вариант может быть выбран только 1 раз. Выбирайте в следующем порядке.

<b>Если дислоцирован Дроид-разведчик:</b>
Выберите вариант 3.

<b>Если есть по крайней мере два героя в пределах 3 клеток от двери:</b>
Выберите вариант 1.

<b>Если нельзя выбрать ничего из вышеперечисленного:</b>
Выберите вариант 4.

<b>Если нельзя выбрать ничего из вышеперечисленного, а вариант 4 уже был выбран:
Выберите вариант 2.

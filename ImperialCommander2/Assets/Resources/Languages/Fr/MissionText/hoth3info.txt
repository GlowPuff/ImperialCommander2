<b>Règles Spéciales:</b>
Lors d'un déplacement les patrouilles de droïdes se déplacent de 4 vers le Rebelle le plus proche.

<b>Stratégie de Repositionnement:</b>
Bloquer l'accès aux terminaux.

<b>Victoire Rebelle:</b>
Quand tous les terminaux ont été appropriés.

<b>Victoire Impériale:</b>
Quand tous les héros sont blessés.

<b>Quand deux terminaux ont été appropriés:</b> <color="red">Patrouilles Accrues</color>

<b>Chaque fois qu'une porte s'ouvre:</b> <color="red">Au Sein du Complexe</color>. Choisir les options dans l'ordre suivant: 2, 1, 3.

<b>Quand le terminal vert est approprié:</b> <color="red">Terminal Vert</color>

<b>Quand le terminal jaune est approprié:</b> <color="red">Terminal Jaune</color>

<b>Quand le terminal bleu est approprié:</b> <color="red">Terminal Bleu</color>

<b>Quand le terminal rouge est approprié:</b> <color="red">Terminal Rouge</color>
